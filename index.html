About Me

Crafting Scalable Solutions

I'm a Software Developer at the National Informatics Centre (NIC), Gandhinagar, where I design and optimize backend systems for E-oLakh a gujarat government CRS portal. With a Computer Engineering degree from Silver Oak University (CGPA 9.41), I bring both academic excellence and hands-on expertise to every project.

My journey includes 2 years as a Software Developer Intern at Sapphire Software Solutions, where I contributed to the development of "Vidyalya" gaining deep exposure to the complete software development lifecycle.

I specialize in .NET Core architecture, MS SQL Server optimization, and building high-performance APIs with clean, maintainable code. My approach combines structured thinking with performance tuning to deliver scalable backend solutions. I work extensively with ADO.NET, Entity Framework Core, LINQ optimization, and JWT-based authentication systems.

Currently expanding my skillset into the frontend with JavaScript and React, I'm passionate about building practical, real-world digital products. My technical toolkit includes GitHub for version control, and VS Code as my development environment of choice.

I believe in writing code that not only works but performs exceptionally under pressure, because in backend development, reliability and speed aren't optional, they're essential.
